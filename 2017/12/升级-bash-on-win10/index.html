升级 bash on win10

  2017-12-13   Study Notes   Web Linux 字数统计: 55字 阅读时长: 1分

通过命令行可以直接将 bash on win10 从14.04LTS 升级到 16.04LTS：

cat  /etc/issue   #确认系统当前版本
sudo apt update
sudo do-release-upgrade  #版本更新
